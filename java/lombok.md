## 加入maven dependency
```
<dependency>
    <groupId>org.projectlombok</groupId>
    <artifactId>lombok</artifactId>
    <version>1.18.12</version>
</dependency>
```
## 在Intellij IDEA 安裝lombok插件
Preferences -> plugins
搜尋 lombok
